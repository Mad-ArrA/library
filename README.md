
### Установка (frontend & backend)

```
npm install
cd frontend
npm install
```

### Компиляция

```
# frontend (:3000) & backend (:5000)
npm run dev

# backend 
npm run server
```

## Build & Deploy

```
cd frontend
npm run build
```

### БД


```
# Импорт бд
npm run data:import

# Удаление бд
npm run data:destroy
```
